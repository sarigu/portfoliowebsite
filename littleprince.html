the little prince

about the task

The project was about creating an interactive story with focus on the story, a style, character design, animations and interactivity design for a specific target group. I was part of the coding team.

What I did

the inspiration

We wanted to create an animation based on „The Little Prince“. It is an animation for young adults who are interested in art. We were given a Jimmy Hendrix Song called "..And The Gods Made Love".

Listen to the song

The design

The style we had to use was Surrealism.

The story + interactivity

We decided on the scene, when the little prince meets the fox for the very first time while still having his rose in mind. Interactivity is a central part of the story. Without user clicks the story wouldn't continue. Besides that we incorporated four eastereggs. Click on the images below to see them.

What I have learned

I experienced really good group work. It was nice to see others code and learn from it. I understood how to make a sequenz diagram. In the beginning it was confusing to work on the same code at the same time, but we learned to cope with it and how important it is to maintain a clean strucure, share changes, talk about names and splitting up part, that need to be coded, in a smart way. It was also interesting to see what users expect from clicking on interactive elements.
