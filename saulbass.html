Task 01.04.01
Basic Web

Saul Bass (Organic Design)

about the task

The assignment was to create a responsive website. The website as well as the design should be based on a given style. As we worked on a mobile version of the website berforehand we were told to add grid layouts, media queries and other elements to it to turn it into a website, that can be accessed from multiple devices.

What I did

The design

My given style was Saul Bass' Organic Design. I began the project by researching the given style and generate ideas from it. This is what the final product looks like.

Font

I used a san serif font with wide and crooked letters, which creates a cut out/handemade feel. When used with only capital letters it resembles the font often find on posters made by Saul Bass.


Colors

One can find a simple use of color in Organic Design. One bold and vivid color was used for backgrounds, negative space and/or lettering, whereas black and whites are often used for the simple shapes and/or lettering.

Images with backgrounds

The decision of colors is influenced by colors used in renown posters (see above). I added the shapes to to give it a paper cut out feel.

Logo

The logo relates to the domainname: Guckiemonster (a combination of the „cookiemonster“ and the fact that people tend to misread my surname „Guci“ as „Guckie)“. The monstereye coming out of a hole represents curiosity. The letters are inspired by cookies.

Banner

Since Saul Bass did a lot of movie posters and film credits, I wanted to introduce a feature that looks like a very simple version of a usual „organic design“ - posters. I added crooked lines, a white outline, a bold color and an organic font.

What I have learned

Since it was my first time making a responsive website, I have learned and got introduced to quite a lot of things:

  • The basics of CSS and HTML
  • Basic image processing (Photoshop)
  • Basic vector drawing (Illustrator)
  • Using XD and paper to create prototypes
  • How to change a layout according to the device its shown on
  • How to set up a responsive project (starting from a mobile version)
  • The process of designing from coming up with an idea, sketching it, applying design principles, improving ideas to developing the final result
