Simple calculator

about the task

It was the first day of the project pool. The task was to create a simple calculator. It should feel good, look good and work.

What I did

The code

Step 1

I began the task by making two input fields and a button labeled „add numbers“, which returned the sum of the numbers typed into the fields when clicked.

Step 2

I then added 3 more buttons (one for each mathematical operation left) and returned the result accordingly.

Step 3

Finally, I removed one input field added a button labeled „=„ and made the calculator work like a „typical“ calculator. click here to find out more about the code.

The look

I styled all the elements afterwards.

Have a look at the code on GitHub

The final version

Calculate some numbers here.

What I have learned

Throughout the project I had a „typical“ calculator as an end goal in mind. Since I couldn’t think of a way to do it, I started by coding the calculator whithout any of the suggested features. So I learned how to break down a big, „unsolvable“ task into smaller, more managable pieces.
