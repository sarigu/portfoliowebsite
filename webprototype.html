User test

I used a "Think aloud" test

  • Please go to the chat-page
  • Find a new message among the chats
  • Find less recent chat among group chats
  • Write someone new
  • Find most recent chat with a fellow student and write message
  • Make a group and name it
  • Share files in group chat
  • Add group members
  • Chat with an individual group member

I got positive feedback as well as suggestions for future enhancements: Adding a group member is not easy enough to spot out. Naming the group looks like renaming. The experts weren't expecting that one can write a new message by clicking on the pop up. The experts suggested that a time or date on the messages could be nice and also a message icon on top.

Technical test

The code is validated. Regarding the perfomance: The app works differently on different browsers. On some browsers the positioning of elements seems off whereas it is in place on other browsers. Also some animations (the chats messages or the pop up text on the message-page) could be improved.
